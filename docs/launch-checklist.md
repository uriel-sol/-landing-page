# צ'קליסט להשקה — Launch Checklist

רשימה מלאה של פריטים שצריכים להיות מוכנים לפני השקת הדף.
מנוהל על ידי qa-launch, מאושר סופית על ידי architect-lead.

---

## תשתית

- [ ] Source of truth (קובץ הדף האחרון) נמצא בריפו
- [ ] מבנה קבצים מסודר ונקי
- [ ] Domain מוגדר
- [ ] Hosting מוגדר
- [ ] SSL certificate פעיל

## תוכן

- [ ] כל הקופי סופי ומאושר
- [ ] תמונות סופיות במקום
- [ ] לוגו סופי במקום
- [ ] Favicon מוגדר
- [ ] OG image מוגדר ונבדק

## טכני

- [ ] Responsive — נבדק על mobile, tablet, desktop
- [ ] Cross-browser — Chrome, Firefox, Safari, Edge
- [ ] ביצועים — Lighthouse score מעל 90
- [ ] נגישות — ניגודיות, keyboard navigation, ARIA
- [ ] טעינה מהירה — תמונות אופטימיזציות, lazy loading
- [ ] קוד נקי — אין console errors, אין dead code

## טפסים

- [ ] טופס Lead עובד (שם, מייל, טלפון, מחזור)
- [ ] Webhook מחובר ל-Make.com
- [ ] ולידציה עובדת (מייל, טלפון)
- [ ] הודעת הצלחה מוצגת
- [ ] נתונים מגיעים ליעד

## אנליטיקס

- [ ] Google Analytics מוגדר ופעיל
- [ ] Facebook Pixel מוגדר ופעיל
- [ ] Events מוגדרים (form submit, CTA click)
- [ ] נבדק שהנתונים מגיעים

## SEO

- [ ] Title tag סופי
- [ ] Meta description סופי
- [ ] OG tags מלאים
- [ ] Canonical URL מוגדר

## סופי

- [ ] QA review מלא בוצע
- [ ] architect-lead אישר מוכנות
- [ ] המשתמש אישר להשקה
- [ ] Backup של הגרסה הסופית

---

> **סטטוס נוכחי:** טרום-פיתוח — ממתין להוספת קובץ הדף לריפו
